Project Overview

This project focuses on the RTL design and verification of Finite State Machines (FSMs) using Verilog HDL. FSMs are fundamental building blocks of digital systems and are widely used for implementing control logic in hardware designs.

The project includes:

- Moore FSM Design
- Mealy FSM Design
- Traffic Light Controller
- Sequence Detector for 1011
- Testbench Development
- Functional Verification
- Waveform Analysis

---

Tools Used

- Verilog HDL
- Vivado Simulator
- Icarus Verilog
- GTKWave
- EDA Playground

Implemented Designs

- GREEN

2. Sequence Detector (Mealy FSM)

Pattern Detected:

1011

- Sequential Pattern Detection
- State Transition Verification
- Output Generation upon Detection

Verification

The functionality of each FSM was verified using testbenches and simulation waveforms. The generated waveforms confirmed proper state transitions and expected output behavior.

---

Results

The Moore FSM Traffic Light Controller and Mealy FSM Sequence Detector were successfully implemented and verified. Simulation results confirmed correct operation of all designed modules.
